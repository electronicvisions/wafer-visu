Module pixiBackend

A backend to draw graphics on a HTML 5 canvas using the library PixiJS. The backend includes functions to draw primitive shapes either as graphics objects or as sprites with a specified resolution. Functions to store multiple primitive shapes of one type as one PixiJS graphics object are also included.
