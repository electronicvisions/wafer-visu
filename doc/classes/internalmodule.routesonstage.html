Class RoutesOnStage

Controls all the routes that are visualized. All routes are stored as a new instance of the class Route. When a route is selected, an additional route is drawn on top of all the other routes, so selected route are not hidden by other routes. When the class is constructed with a routes array, the number of positions (first index) have to match the number of routes.
